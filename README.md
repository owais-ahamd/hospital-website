# Hospitel Website (General Hospital)

A simple multi-page hospital website built using **HTML** and **CSS**.

## Pages
- `home.html` – Home / landing page
- `About.html` – About / Vision & Mission
- `services.html` – Services
- `contact.html` – Contact form + embedded map

## Styling
Each page links to its own stylesheet:
- `home.css`
- `about.css`
- `services.css`
- `contact.css`

## Assets
- `logo.svg`

## How to Run
Open any of the HTML files in your browser (no build step required).

Example:
- `home.html` (then navigate using the header links)

## Notes
- The site is static (no backend).
- The contact page uses a Google Maps embed iframe.
